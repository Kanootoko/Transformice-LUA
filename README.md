# Transformice-LUA
LUA script(s) for triberoom.
<hr>
MainScript
-----------------------------------
Управление:
###
'o' / "Facepalm" - превращение в шамана<br>
Эмоция "Anger" - превращение в вампира (визуальный эффект), спадает при смерти<br>
'p' - спирит под мышь <br>
'Shift' - телепортироваться на ~50 пикселей вверх<br>
'8' на доп. клавиатуре - возродиться<br>
'9' на доп. клавиатуре - сменить цвет ника над мышью на случайный<br>
<br>
'Tab' - открытие работы со спавном по клику мыши: <br>
--> '['  - выбор синего портала<br>
--> ']'  - выбор оранжевого портала<br>
--> 'b'  - выбор пузыря<br>
--> 'F9' - выбор удаления объекта (удаляет с маленького расстояния от центра, некоторые типы объектов имеют центр в каком-то другом месте, и их сложно удалить)<br>
--> Чат-команда "!spawn [id объекта]" (id можно узнать, например, здесь: <http://atelier801.com/topic?f=6&t=466450&p=1>) - выбор объекта по номеру<br>
--> Чат-команда "!ghost" - включение/выключение режима создания призрачных объектов<br>
--> Чат-команда "Angle [x]" - установление начального угла поворота объекта (абсолютного или суммируемого при спавне в два клика)
--> Чат-команда "!2steps" - включение/выключение работы спавна в два клика: Первый - место появление объекта, второй - направление и скорость, скоторыми объект будет создан<br>
----> Чат-команда "!Speed [x]" - установление начальной скорости объекта
----> Чат-команда "!AbsoluteAngle" - включение/выключение абсолютности изначального угла поворота (изначально false)

'`' ('~') - закрытие работы со спавном, после клика ничего происходить не будет<br>
<br>
'F4' - включение телепортации по клику (на 1 раз)<br>
Чат-команда "!clear" - очистка области от объектов (кроме созданных на карте до запуска скрипта)<br>
Чат-команда "!res" - запомнить данное местоположение мыши как точку воскрешения<br>
Чат-команда "!res null" - поставить точку воскрешения в станартном месте для карты<br>

Чат-команда "!AutoNewGame" - переключает, начнется ли новая игра по истечению времени (изначально false)
Чат-команда "!AutoNewShaman" - переключает, будет ли в начале раунда шаман назначен по умолчанию (изначально false)
Чат-команда "!afkDeath" - переключает, будут ли умирать мыши через 20секунд от начала раунда, не двигаясь (изначально false)
